使用说明

基本用法

在模板文件中使用以下结构创建分页：

<nav aria-label="分页">
    <ul class="pagination">
        <li class="page-item"><a class="page-link" href="#">1</a></li>
        <li class="page-item active"><a class="page-link" href="#">2</a></li>
        <li class="page-item"><a class="page-link" href="#">3</a></li>
    </ul>
</nav>
样式变体

添加以下类名来使用不同的样式变体：

  • pagination-rounded - 圆形分页按钮
  • pagination-minimal - 简约风格分页
  • pagination-shadow - 带阴影的分页
分页区域

使用 pagination-section 类创建带装饰线的分页区域：

<div class="pagination-section">
    <ul class="pagination">
        ...
    </ul>
</div>
尺寸变体

使用 pagination-lg 或 pagination-sm 类调整分页尺寸。

对齐方式

使用 pagination-center 或 pagination-end 类调整分页对齐方式。
